## Developers

* Install: `npm install` and `bower install`
* Run locally: `gulp serve`
* Watch files + automatically run tests: `gulp watch`
* Build: `gulp build`
* Deploy to gh-pages: `gulp deploy`

### Testing

* `gulp test`
